## 机器学习算法的分类

### 监督学习算法
#### 概念
根据我们提供的训练集中的**输入和输出**来推断出最符合训练集的模型，然后根据模型来预测后续输入对应的输出值的一种算法，主要有两种模型：回归(regression)和分类(classcification)
#### 分类
1. 回归算法：用于预测数值的算法，并且这个预测的数值的范围是近乎无穷的
2. 分类算法：用于预测数值或非数值的算法，这个预测的范围是有限的

### 无监督学习
#### 概念
根据我们所提供的训练集中的数据来寻找数据之间的关系的算法，与监督算法最大的区别在于训练集并不会有所谓的“right answer”。
