## 监督学习算法的流程
1. 输入训练集，训练集要包含正确的输入(input)和输出(output)
2. 根据算法以及训练集，得到一个合适的模型
3. 根据所得到的模型，可以得到根据输入预测得到的输出

![alt 监督学习算法的流程](https://bear-img-1319754387.cos.ap-guangzhou.myqcloud.com/website/machine%20learning/working%20process.png)


## 监督学习算法的分类
1. 回归学习算法(regression)
2. 分类学习算法(classification)
